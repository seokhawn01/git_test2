# GDGoc 그로우톤 6팀 (iOS) Travodo 백엔드

Travodo 백엔드 서버 레포지토리입니다.

---

## 📌 커밋 메시지 컨벤션

- **Feat/#이슈번호** : 새로운 기능 추가  
- **Fix/#이슈번호** : 버그 수정  
- **Chore/#이슈번호** : 빌드 작업, 설정 변경  
- **Refactor/#이슈번호** : 코드 리팩토링 (기능 변경 없음)  
- **Docs/#이슈번호** : 문서 수정  

---

## 🌿 PR & 브랜치 전략

1. 이슈 생성  
   - 예) `[FEAT] 로그인 구현`

2. 이슈 번호로 브랜치 생성  
   - 예) `feat/#1`

3. 해당 브랜치에서 작업 후  
   - `commit` → `push` → `PR 생성`

4. **1명 이상 코드 리뷰 승인 후**  
   - `main` 브랜치로 merge

> ❗ **PR 시 CI가 깨지면 merge 금지**

---

## ✅ PR 체크 사항

- CI 통과 여부 확인
- 충돌(conflict) 없음 확인
- 리뷰 승인 여부 확인
